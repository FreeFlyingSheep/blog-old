通过重定向让 shell 切换环境后继续执行。

<!--more-->

## 用途

在执行 `su`、`exec` 或者 `chroot` 之类的指令后，继续执行想要的命令，主要是在尝试写自动构建 LFS 的时候遇到的。

## 实例

## `su`

```bash
su - lfs << "EOF"
<command>
EOF
```

## `exec`

```bash
exec /bin/bash -c "<command>"

exec /bin/bash << "EOF"
<command>
EOF

```

## `chroot`

```bash
chroot /mnt/lfs /bin/bash -c "<command>"

chroot /mnt/lfs /bin/bash << "EOF"
<command>
EOF
```
